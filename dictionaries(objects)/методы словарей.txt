Краткое описание методов.

dict.items( ) - возвращает объект типа dict_items, состоящий из списка
кортежей пар ключ-значение

dict.keys( ) - возвращает объект типа dict_keys, состоящий из списка
ключей словаря

dict.values( ) - возвращает объект типа dict_values, состоящий из списка
значений словаря

dict.clear( ) - удаляет все элементы словаря

dict.pop( ) - удаляет из словаря запрашиваемый ключ и возвращает его
значение

dict.popitem( ) - удаляет пару ключ-значение и возвращает кортеж,
состоящий из этой пары

dict.copy( ) - возвращает копию словаря

dict.get( ) - возвращает значение запрашиваемого ключа, если в словаре
нет запрашиваемого ключа, то возвращает или значение
второго аргумента или None

dict.setdefault( ) - возвращает значение запрашиваемого ключа, если в
словаре нет запрашиваемого ключа, метод добавляет
ключ и присваивает ему, в качестве значения или
значение второго аргумента или None

dict.fromkeys( ) - возвращает словарь в качестве ключей которого
используются значения аргумента.

dict.update( ) -обновляет словарь добавляя пары ключ-значение или
заменяя значение ключа.
