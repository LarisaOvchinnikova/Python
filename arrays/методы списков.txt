Краткое описание методов.

list.append( ) - вставляет в конец исходного списка значение аргумента.

list.extend( ) - вставляет в конец исходного списка все значение
аргумента (итерируемого объекта)

list.insert( ) - вставляет в список значение аргумента перед элементом
индекс которого равен значению индекса переданного
методу в качестве аргумента.

list.remove( ) - удаляет первый элемент списка равный аргументу

list.рор( ) - удаляет и возвращает элемент списка с индексом равным
аргументу метода или удаляет последний элемент списка если
аргумент метода не указан

list.clear( ) - удаляет из списка все элементы

list.index( ) - возвращает индекс первого элемента списка равного
аргументу

list. sort ( ) - сортирует список

list.reverse( ) - переставляет элементы в обратном порядке

list.copy( ) - возвращает копию списка

list.count( ) - возвращает число вхождений аргумента метода в список
